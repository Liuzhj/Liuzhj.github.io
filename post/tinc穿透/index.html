VPS 服务端部署

  • 使用的是搬瓦工 VPS，机器还是借用同事的，哈哈 ……

    项目 配置
    CPU 双核
    内存 1 G
    操作系统 OS CentOS 7

  • 安装软件包，可以从epel源中直接安装 tinc

    1
2

    yum install epel-release -y
yum install tinc -y

  • 配置软件目录

    1

    mkdir -p /etc/tinc/bwgvps/hosts

  • Tinc 配置文件

    /etc/tinc/bwgvps/tinc.conf

    1
2
3
4
5

    Name=bwgvps
Interface=vpn
Mode=switch
Cipher=aes-256-cbc
Digest=sha512

  • 服务端 IP + 公钥配置文件

    /etc/tinc/bwgvps/hosts/bwgvps

    1
2

    Address = 97.64.45.124
Subnet = 10.0.0.1/32

  • 生成 Tinc 公私钥文件

    1

    tincd -n bwgvps -K4096

  • 生成 Tinc 管理服务脚本

    1
2
3
4
5
6
7

    cat /etc/tinc/bwgvps/tinc-up
#!/bin/sh
ifconfig $INTERFACE 10.0.0.1 netmask 255.255.255.0
  
cat /etc/tinc/bwgvps/tinc-down
#!/bin/sh
ifconfig $INTERFACE down

  • 对管理服务脚本添加执行权限

    1

    chmod +x /etc/tinc/bwgvps/tinc-*

  • 开启 IP 地址转发

    1

    echo "net.ipv4.ip_forward = 1" >> /etc/sysctl.conf && sysctl -p

  • CentOS 7 启停服务命令

    1
2

    systemctl start tinc@bwgvps
systemctl enable tinc@bwgvps

  • 服务日志

    1

    tail -f /var/log/message
