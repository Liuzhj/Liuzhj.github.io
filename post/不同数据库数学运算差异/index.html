原因是啥

那为啥 SqlServer, PostgreSQL 和 Python 2.X 的结果不正确呢？ 原因出在这里

1
2

((5026-4700-250)/100 * 0.0290)
(5026-4700-250)/100

以上运算在 SqlServer, PostgreSQL 和 Python 2.X 结果为 0，真实的原因是它们执行的是整数除法，而不是浮点除法

那怎样让他们执行浮点除法呢？很简单，只要在运算前声明为 floating-point divison 即可

1
2
3

((5026-4700-250)/100. * 0.0290)
或者
((5026-4700-250)/100.0 * 0.0290)
